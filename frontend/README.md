# Todo Frontend (React + Vite + Tailwind)

## Features
- Auth UI (Login/Signup)
- Protected Routes
- Role-Based UI (Admin Dashboard)
- Todo Management (CRUD + Pagination)
- Profile Management
- TanStack Query for state management

## Setup
1. `npm install`
2. `npm run dev`

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- TanStack Query
- Axios
- Lucide React
